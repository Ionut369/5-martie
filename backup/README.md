# Website Backup - Apollo Tech

This backup was created on February 20, 2025.

## Project Structure

The backup contains the following key components:

1. React Application
2. Tailwind CSS Styling
3. i18n Translations (Romanian and English)
4. Supabase Integration
5. Component Library
6. Page Components
7. Utility Functions

## Key Features

- Multi-language Support (RO/EN)
- Responsive Design
- AI Chatbot Integration
- Newsletter Subscription
- Contact Forms
- Legal Pages
- Accessibility Features

## Technology Stack

- React 18.3.1
- Vite
- TypeScript
- Tailwind CSS
- i18next
- Supabase
- Lucide React Icons

## Deployment

The site is deployed on Netlify with automatic deployments from the main branch.

## Important Notes

1. All CTA buttons are linked to Calendly
2. The chatbot is integrated via Chatbase
3. Newsletter subscriptions are stored in Supabase
4. The site supports both Romanian and English languages with automatic detection
